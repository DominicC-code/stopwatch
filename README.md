# Stopwatch

A desktop stopwatch application built with PyQt6 featuring keyboard shortcuts and a custom red theme.

## Features:
- Start, stop, and reset functionality
- Precise timing down to milliseconds
- Keyboard shortcuts for quick control
- Custom font (Nordhin)
- Clean red-themed interface

## Keyboard Shortcuts:
- **Space** - Start the stopwatch
- **T** - Stop the stopwatch
- **R** - Reset to 00:00:00:00

## How to use:
1. Install PyQt6: `pip install PyQt6`
2. Download the Nordhin.otf font file (or change to any font you have)
3. Run `python stopwatch.py`
4. Use buttons or keyboard shortcuts to control the timer
